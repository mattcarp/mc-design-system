Design Principles

Core Guidelines (AKA Matt's Design Neuroses)

  • No Linear Gradients: Because Matt thinks they're tacky and this is his system
  • Typography: Obsessively thin font weights (100-300) because elegance > readability, apparently
  • Colors: Dark theme with blue and purple accents (Matt's favorite colors, shocking)
  • Professional: Clean, modern, minimalist aesthetic (translation: Matt removed all the fun parts)
  • Consistency: Single source of truth for all design decisions (because Matt can't handle chaos)
